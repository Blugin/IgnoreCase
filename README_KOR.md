## :wrench: 설치
1) 릴리즈 페이지에서 `.phar`을 다운로드하세요
2) 다운로드한 `.phar`파일을 서버의 **/plugins/** 폴더에 넣으세요  
3) 서버를 다시 시작하거나, `/reload` 명령어를 실행하세요
